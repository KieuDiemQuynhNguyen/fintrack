# FinTrack — Personal Finance PWA

## Deploy to GitHub Pages (free)

1. Create a GitHub account at github.com
2. Create a new repository called `fintrack` (make it public)
3. Upload all files from this folder to the repository
4. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
5. Your app will be live at: `https://yourusername.github.io/fintrack`

## Install on iPhone
1. Open your GitHub Pages URL in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"
5. FinTrack now appears on your home screen like a native app

## Features
- Dashboard: Net worth, portfolio overview, recent transactions
- Portfolio: Holdings tracker with gain/loss, donut chart, 15-year projection
- Budget: Monthly income vs expenses, income allocation sliders
- Goals: Savings goals with progress bars
- Settings: Profile, account balances, income setup

## Your data
All data saves automatically to your browser's local storage.
It persists between sessions on the same device/browser.
